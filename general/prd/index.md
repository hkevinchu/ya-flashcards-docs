# Product Requirements (A1)

## Summary
Flashcard app for iOS/iPadOS. Users create decks and cards, study them, and their content syncs via iCloud.

## Target users
- General public
- Family members (but each user has private iCloud data)

## Core user journeys (MVP)
1. Create a deck
2. Add cards to a deck
3. Study a deck (reveal answer, grade)
4. Browse bundled decks and duplicate into My Decks

## Screens (MVP)
- My Decks (list + empty state)
- Deck Detail (cards list + add/edit)
- Study (question/answer + grading)
- Bundled Decks (read-only list + duplicate)

## Definition of done (MVP)
- All MVP journeys work offline and sync when iCloud is available
- No crashes in common flows
- Basic accessibility labels present on primary controls
