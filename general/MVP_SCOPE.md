# MVP Scope (A1)

## In scope (MVP)
- [x] iCloud identity only (no in-app login)
- [x] SwiftData + iCloud sync for user-created content
- [x] iOS and iPadOS support (latest OS only)
- [x] Offline-first behavior (works without network, syncs when available)
- [x] Portrait orientation supported (landscape optional, not optimized)

- [x] Decks: create, rename, delete
- [x] Decks may exist with zero cards

- [x] Cards: create, edit, delete
- [x] Cards display in creation order (no manual reordering in MVP)

- [x] Study mode: show question, tap to reveal answer, user marks Correct or Incorrect

- [x] Bundled starter decks: read-only, duplicable into My Decks
- [x] Bundled decks are always visible and cannot be deleted
- [x] Bundled decks may be duplicated multiple times into My Decks

- [x] Basic empty states (no decks / no cards)

## Out of scope (Post-MVP)
- [x] Cross-user sharing (family shared decks across Apple IDs)
- [x] Accounts / Sign in with Apple
- [x] Import/export (CSV, Anki, etc.)
- [x] Tags, search, reminders
- [x] Spaced repetition algorithm beyond simple grading
- [x] Card media (images, audio, video)
- [x] Manual card reordering
- [x] Cloud admin tools / web portal

## Open questions (must be zero before leaving A1)
- [x] None

- [x] Out-of-scope list is final
- [x] Platform support is final (latest only)
